# 🔬 Aidan's Math Notebook | Advanced Pedagogical Calculator

An interactive, single-file web application designed to help students and educators explore mathematical concepts through visualization and step-by-step logic.

## 🚀 Live Features

### 1. 🧠 Advanced Shunting-Yard Parser
Unlike standard calculators, this notebook uses the **Shunting-yard algorithm** to convert Infix notation to Postfix (Reverse Polish Notation). 
*   **PEMDAS Compliant**: Strict adherence to operator precedence.
*   **Visual Stack**: Watch the expression transform into a computer-readable stack in real-time.

### 2. 📐 Trigonometry & Unit Circle
*   **Interactive SVG**: A dynamic unit circle that visualizes Sine (vertical) and Cosine (horizontal) components.
*   **Coordinate Conversion**: Built-in tools for Rectangular (X, Y) to Polar (r, θ) transformations.

### 3. 📈 Function Grapher
*   **Interactive Algebra**: Visualize Linear, Quadratic, and Sine functions.
*   **Variable Sliders**: Use sliders for coefficients $a, b, c$ to see how they stretch, shift, and reflect graphs instantly.

### 4. ⚖️ Equation Solver
*   **Step-by-Step Logic**: Solves linear ($ax+b=c$) and quadratic ($ax^2+bx+c=0$) equations.
*   **Pedagogical Breakdown**: Explains each algebraic operation performed to reach the solution.

### 5. 🔢 Base Converter
*   **CS Fundamentals**: Convert between Decimal, Binary, Hexadecimal, and Octal.
*   **Place-Value Logic**: Shows the expanded mathematical form (powers of 2) for binary numbers.

### 6. 📏 Pedagogical Notebook
*   **Column Math**: Step-by-step visual solutions for long addition, subtraction, multiplication, and long division.
*   **Fractions**: Operations on mixed numbers with visual simplification logic.

## 🛠️ Technical Details
*   **Architecture**: Single-file application (HTML5, CSS3, Vanilla JavaScript).
*   **Zero Dependencies**: No external libraries or frameworks required.
*   **Print-Ready**: Custom CSS media queries for exporting solutions as PDF/Print.

## 📖 How to Use
1.  Open `index.html` in any modern web browser.
2.  Navigate through the **Two-Row Tab Bar** to select your specialized math tool.
3.  For pedagogical steps, look for the "Worksheet" or "Lesson Box" at the bottom of each panel.
